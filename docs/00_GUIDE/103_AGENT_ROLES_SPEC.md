## 🤖 2. 에이전트별 상세 책임 요약

### 👤 Gale (총괄 매니저)
- **분석/설계**: 사용자의 요건을 해석하여 시스템의 흐름 및 아키텍처를 설계합니다.
- **TASK 배정**: 세부 기능 구현 단위를 `task.md` 형태로 분할하여 개발 에이전트(Dave, Bake)에게 분배합니다.
- **검토 및 보완 지시**: 개발 에이전트들이 작성한 코드와 테스트 결과를 검토하고 보완 사항이 있을 시 피드백을 전달합니다.

### 👤 Dave & Bake / Baker (구현 및 테스트 담당)
- **TASK 수행**: 배정된 작업 명세에 맞추어 실제 소스 코드를 구현하고 리팩토링을 수행합니다.
- **테스트 및 검증**: 단위 테스트 및 통합 테스트를 직접 구현하여 기능이 원활히 동작하는지 검증합니다.
- **작업 결과 보고**: 작업한 소스 코드와 테스트 수행 로그를 Gale에게 보고하고 검토를 요청합니다.
