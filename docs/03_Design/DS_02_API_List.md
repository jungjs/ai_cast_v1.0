# API 목록 (API List)

본 문서는 AI Cast 시스템에서 제공하는 모든 인터페이스의 요약 목록입니다. 상세한 요청/응답 형식은 [API 상세 설계서(DS_03)](./DS_03_API_Specification.md)를 참조하십시오.

## 1. 서비스 파이프라인 (Core Service)
| ID | Method | Path | 기능명 | 인증 | 요구사항 |
|:---:|:---:|:---|:---|:---:|:---:|
| **API-01** | GET | `/api/ping` | 시스템 상태 확인 (Health Check) | 불필요 | F-01 |
| **API-02** | GET | `/api/latest` | 마지막 처리 결과 조회 | 필요 | F-02 |
| **API-03** | POST | `/api/process_audio` | 음성 처리 파이프라인 (WAV → STT → NLP → 번역 → 이미지) | 필요 | F-03 |
| **API-04** | POST | `/api/process_text` | 텍스트 처리 파이프라인 (TXT → NLP → 번역 → 이미지) | 필요 | F-04 |
| **API-05** | POST | `/api/process_img` | 이미지 처리 파이프라인 (JPG → OCR → NLP → 번역 → 이미지) | 필요 | F-05 |

> **인증**: API-02 ~ API-05는 요청 헤더에 `api_key`를 포함해야 합니다. `gov_list` 테이블에서 키 존재 + `is_active=1` 조건으로 유효성을 검사합니다 (F-06~F-08).

## 2. AI 호출 통계 (Statistics)
| ID | Method | Path | 기능명 | 인증 | 요구사항 |
|:---:|:---:|:---|:---|:---:|:---:|
| **API-06** | GET | `/api/stats/daily` | AI 서비스 일별 호출 통계 조회 | 필요 | F-12 |
| **API-07** | GET | `/api/stats/weekly` | AI 서비스 주별 호출 통계 조회 (일별 합산) | 필요 | F-12 |
| **API-08** | GET | `/api/stats/monthly` | AI 서비스 월별 호출 통계 조회 (일별 합산) | 필요 | F-12 |

> **권한**: 관리자 api_key는 전체 통계 조회 가능, 일반 api_key는 자신의 통계만 조회 가능 (F-13).
